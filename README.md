# plain

Colorful git-aware plain text prompt

<p align="center">
<img src="https://raw.githubusercontent.com/changyuheng/fish-theme-plain/master/screenshot.png">
</p>

## Features

* No need to use a patched font
* Colorful
* Not folding path
* Detecting remote logged in and UID
* Integrates Git

## Install

With [Fisherman]

```
fisher i plain
```
